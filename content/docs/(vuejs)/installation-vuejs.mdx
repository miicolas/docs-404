Vue.js offre plusieurs options de personnalisation :

- **Vue CLI** : Utilisez `vue.config.js` Ã  la racine du projet

```js

  plugins: [vue()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src')
    }
  },
  server: {
    port: 3000
  }
})
```
