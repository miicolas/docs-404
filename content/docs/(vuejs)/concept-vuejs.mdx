### C'est quoi Vue.js ?

**Vue.js** est comme un chef d'orchestre 🎭 pour votre interface utilisateur :

Vous déclarez ce que vous voulez voir **(template)** → Vue réagit aux changements de données → L'interface se met à jour automatiquement **(rendu réactif)**.

**Exemple concret :**

```html
<div id="app">
  <p>Compteur : {{ compteur }}</p>
  <button @click="incrementer">Augmenter</button>
</div>

<script>
  Vue.createApp({
    data() {
      return {
        compteur: 0
      }
    },
    methods: {
      incrementer() {
        this.compteur++
      }
    }
  }).mount('#app')
</script>
```

**Cas d'usage réel :**
Une application de liste de tâches qui réagit instantanément aux interactions utilisateur :
```html
<div id="todo-app">
  <input v-model="nouvelleTache" @keyup.enter="ajouterTache">
  <ul>
    <li v-for="tache in taches" :key="tache.id">{{ tache.texte }}</li>
  </ul>
</div>
```
